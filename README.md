# Sistema de Adoção de Pets

Projeto desenvolvido a partir do desafio **Sistema de Cadastros**, proposto por Lucas Carrilho.

Aplicação em Java voltada à prática de lógica de programação e organização de código,
com execução via terminal.

## Tecnologias
- Java 17
- Git e GitHub
